## Descrição 
Este projeto foi criado para praticar os conceitos de branches e merges no Git.

## Como instalar
Acesse o site do GitHub pelo navegador e crie sua conta e ativa ela pelo seu e-mail escolhido.

Pesquise no navegador GitHub Desktop, para facilitar trabalhar com git e github no computador. Instale e configure.

## Como usar
Pré-Requisito: ter uma conta ativa no github e github desktop instalado e aberto no computador.

Para adicionar uma pasta no github:
- Crie um repositório no GitHub.
- Com a pasta criada e/ou os arquivos prontos digite no terminal com a pasta aberta
git remote add origin <url do repositório>

Para clonar um repositório para GitHub Desktop:
- No repositório do GitHub no navegador, clique em "code". Em "local", copie a url do https.
- No GitHub Desktop, clique em File. Clique em Clone Repository. Na sessão URL, cole a url do repositório que copiou no espaço indicado e e clique no botão Clone.
- Vai mudar a aba para um de carregamento. Depois mudar para a aba do repositório já clonado.

Para puxar os conteúdos do repositório:
- git pull

Para levar os commits e os arquivos adicionados e alterados com:
- git push -u origin <nome_da_branch>
